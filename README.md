Unicorn-UI-Portal
=================

This will be the repo for the Unicorn UI Portal

## Setup

* Install deps at http://harpjs.com/
* harp server --port 9000

## Deploying

```shell
harp compile . ../path/to/deploy/directory
cd ../path/to/deploy/directory
chmod 755 post-deploy.sh
./post-deploy.sh
```
